# promotion-management-system


Trabajo práctico realizado para la UTN - Primer Año Sistemas.

## Descripción

Sistema desarrollado en Python para la administración de:
- usuarios,
- locales,
- promociones,
- descuentos,
- y reportes de uso.

El sistema maneja distintos tipos de usuarios:
- Administrador
- Dueño de local
- Cliente

## Funcionalidades

### Administrador
- Gestión de locales
- Creación de dueños
- Aprobación o rechazo de promociones
- Reportes

### Dueños de locales
- Crear promociones
- Consultar reportes

### Clientes
- Buscar promociones
- Solicitar descuentos

## Tecnologías utilizadas
- Python
- Pickle
- Manejo de archivos `.dat`
- Programación estructurada y orientada a objetos
